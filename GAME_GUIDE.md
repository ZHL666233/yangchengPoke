## 1. 游戏概述

城都宝可梦是一款宝可梦养成/战斗类网页游戏，包含以下核心玩法：

- **初始选择**：从妙蛙种子（草）、小火龙（火）、杰尼龟（水）中选一只开始冒险
- **野外探索**：在8个不同地区探索、战斗、捕捉宝可梦
- **Boss挑战**：每个地区有首领宝可梦，击败后解锁新区域
- **副本爬塔**：挑战无尽层数的副本，层数越深敌人越强
- **养成屋**：喂食、玩耍、打工、训练提升宝可梦能力
- **进化**：通过等级提升或使用进化石让宝可梦进化
- **幸运转盘**：每6小时免费抽一次，有机会获得稀有奖品

## 2. 宝可梦系统

### 2.1 基础属性

每只宝可梦有6项核心属性：

| 属性 | 缩写 | 说明 |
|------|------|------|
| HP | hp | 生命值，归零则无法战斗 |
| 攻击 | atk | 影响物理技能的伤害 |
| 防御 | def | 减少受到的物理伤害 |
| 特攻 | spa | 影响特殊技能的伤害 |
| 特防 | spd | 减少受到的特殊伤害 |
| 速度 | spe | 决定先手顺序（当前版本仅作面板显示） |

### 2.2 属性计算公式

实际属性由 **种族值** + **个体值** + **努力值** + **性格** + **等级** 共同决定。

**HP计算：**
```
HP = floor(((2 × 种族值 + 个体值 + floor(努力值/4)) × 等级) / 100) + 等级 + 10
```

**其他5项属性：**
```
属性 = (floor(((2 × 种族值 + 个体值 + floor(努力值/4)) × 等级) / 100) + 5) × 性格修正
```

- 性格加成的属性 × 1.1
- 性格减成的属性 × 0.9
- 无加成减成的属性不变

### 2.3 个体值（IV）

- **范围**：0 ~ 31（每项独立随机）
- **影响**：直接影响属性面板，31个体值比0个体值多约10%属性
- **闪光宝可梦**：默认6项全部=31（满个体值），显示金色 **6V** 标签
- **提升方式**：使用IV道具（每颗+5，上限31）

### 2.4 努力值（EV）

- **范围**：0 ~ 100（每项独立），总计上限600
- **获取方式**：战斗胜利25%概率获得随机1点EV；副本训练
- **提升方式**：培育屋训练任务

### 2.5 性格系统

共25种性格，其中5种为无影响性格（勤奋、坦率、腓腆、浮躁、认真），其余20种各有+/-一项属性的修正：

| 攻击 | 速度 | 勇敢 |
| 防御 | 攻击 | 大胆 |
| 防御 | 特攻 | 淘气 |
| 防御 | 特防 | 乐天 |
| 防御 | 速度 | 悠闲 |
| 特攻 | 攻击 | 内敛 |
| 特攻 | 防御 | 慢吞吞 |
| 特攻 | 特防 | 马虎 |
| 特攻 | 速度 | 冷静 |
| 特防 | 攻击 | 温和 |

### 2.6 闪光宝可梦

- **出现概率**：野外 1/4096（约0.024%）
- **种族值加成**：闪光宝可梦的种族值额外×1.1（即面板属性额外提升约10%）
- **默认个体值**：闪光宝可梦默认6项IV全部为31
- **捕捉难度**：闪光宝可梦的捕捉率×0.3（更难捕捉）
- **显示**：名字旁显示✨标记

### 2.7 特性系统

每只宝可梦有一个特性：
- **普通特性**：从该物种的特性列表中随机获取
- **隐藏特性**：10%概率获得隐藏特性
- 特性目前为描述性展示，暂无战斗中的主动触发效果

### 2.8 技能系统

- 每只宝可梦最多装备 **4个技能**
- 技能通过升级自动学会，可在技能切换界面更换已学会的技能
- 每个技能有：**威力**、**属性**、**最大PP（使用次数）**
- 战斗中使用技能消耗PP，PP用完则无法再使用该技能
- 治愈队伍时恢复全部PP

### 2.9 等级与经验

- **最高等级**：100级
- **升级所需经验**：`等级 × 50`（如：5→6级需要250经验，99→100级需要4950经验）
- **初始等级**：5级
- **获得经验来源**：战斗胜利、野外事件、使用道具（糖果+40、稀有糖果+120）

## 3. 战斗系统

### 3.1 伤害计算

```
伤害 = floor(((((2 × 攻方等级) / 5 + 2) × 技能威力 × (攻击方攻击 / 防御方防御)) / 50 + 2) × 暴击倍率 × 属性一致加成 × 属性克制倍率 × 随机浮动(0.85~1.0))
```

- **暴击概率**：1/16（约6.25%）
- **暴击倍率**：1.5倍（狙击手特性下为2.25倍）
- **属性一致加成（STAB）**：技能属性与宝可梦属性一致时，伤害×1.5
- **属性克制**：克制×2.0，抵抗×0.5，免疫×0

### 3.3 战斗流程

1. **遇到野生宝可梦**（或Boss）
2. **选择行动**：使用技能攻击 / 切换宝可梦 / 使用道具（精灵球捕捉） / 逃跑
3. **伤害结算**：根据伤害公式计算伤害
4. **击败**：获得经验 + 金币 + 25%概率获得随机1点EV
5. **捕捉机会**：击败后可选择捕捉（HP越低越容易捕捉）
6. **倒下**：自动切换下一只存活的队伍宝可梦，全军覆没则战斗结束

### 3.4 战斗胜利奖励

| 奖励 | 数值 |
|------|------|
| 经验值 | 野生宝可梦等级 × 10 |
| 金币 | 20 ~ 50（随机） |
| 努力值 | 25%概率获得随机1点EV |
| Boss首胜 | 额外500金币 + 解锁下一地图 |

## 4. 捕捉系统

### 4.1 捕捉条件

- 需要先击败野生宝可梦，使其进入虚弱状态
- 然后使用精灵球尝试捕捉
- Boss宝可梦也可以捕捉（击败后进入捕捉阶段）

### 4.2 捕捉概率计算

```
捕捉概率 = floor((HP因子 × 种族值因子 × 等级因子 × 玩家等级加成 × 精灵球倍率 × 闪光因子) / 255 × 100)%
```

各因子计算：

| 因子 | 计算方式 |
|------|---------|
| HP因子 | `(3 × 敌方满HP - 2 × 敌方当前HP) / (3 × 敌方满HP)` — HP越低越容易 |
| 种族值因子 | `max(15, 255 - floor(敌方总属性/2.5))` — 种族值越高越难 |
| 等级因子 | `max(0.5, 1 - (敌方等级-1)/150)` — 等级越高越难 |
| 玩家等级加成 | `1 + 我方等级 × 0.005` — 我方等级越高越容易 |
| 闪光因子 | 闪光×0.3（更难），普通×1.0 |
| 精灵球倍率 | 见下表 |

**精灵球捕捉率倍率：**

| 精灵球 | 倍率 | 价格 |
|--------|------|------|
| 精灵球 | ×1.0 | 50金币 |
| 超级球 | ×1.5 | 150金币 |
| 高级球 | ×1.75 | 300金币 |
| 大师球 | ×∞（100%） | 9,999金币 |

### 4.3 捕捉结果

- 捕捉成功后，宝可梦加入**队伍**或**仓库**（队伍满时自动入仓库）
- 首次遇到该物种时记录在**图鉴**中
- Boss首胜额外奖励500金币 + 解锁下一地图

## 8. 副本系统

### 8.1 基础规则

- 每层遇到一个固定敌人（相同层数始终遇到相同物种）
- 战胜后直接进入下一层，无捕捉环节
- 战斗队伍人数最多6只
- 一键治愈：费用 = 队伍人数 × 30金币（恢复HP+PP）

### 8.2 敌人等级

等级公式（**不随玩家等级变化**）：
```
等级 = max(1, min(100, floor((3 + floor(层数 × 1.5) + 随机(0~4)) × 难度等级倍率)))
```

### 8.3 难度曲线

每10层一个难度段，逐渐增加敌人强度：

| 层数 | 难度 | IV加成 | 每项EV | 等级倍率 |
|------|------|--------|-------|---------|
| 1~10 | 普通 | +0 | 0 | ×1.0 |
